One-time Firebase setup required
1 Go to console.firebase.google.com → Create a project (or use existing)
2 In the project: Build → Firestore Database → Create database (start in production mode)
3 Authentication → Sign-in method → Enable Email/Password
4 Project Settings → Your apps → Add a Web app → Copy the firebaseConfig object
5 Set Firestore rules (Firestore → Rules) to: 
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if request.auth != null;
    }
  }
}
6 Paste your config below and save. Only needs to be done once.
